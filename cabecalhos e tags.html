<b> (Bold)

Função: usada para aplicar negrito ao texto.
Uso semântico: Não adiciona significado adicional ao conteúdo, é puramente para estilização visual.

<strong> (Strong)

Função: usada para destacar um texto, geralmente aplicando negrito.
Uso semântico: Adiciona importância ao conteúdo, indicando que o texto é mais importante ou merece atenção especial.

<i> (Italic)

Função: usada para aplicar itálico ao texto.
Uso semântico: Não adiciona significado adicional ao conteúdo, é puramente para estilização visual.

<em> (Emphasis)

Função: usada para enfatizar um texto, geralmente aplicando itálico.
Uso semântico: Adiciona ênfase ao conteúdo, indicando que o texto é importante ou merece atenção especial.

<a> (Anchor)

Função: usada para criar um hyperlink, ou seja, um link para outra página ou recurso.
Uso semântico: Indica a intenção de que o texto ou elemento seja clicável e leve o usuário para outro local.
Atributo obrigatório: O atributo href especifica o URL de destino para onde o link deve direcionar o usuário.

Isto é um Link para o site do Google

O atributo target especifica onde o navegador deve abrir o link.
Valores Comuns do Atributo target:

_self
* Abre o link na mesma janela ou aba. padrão.

_blank:
* Abre o link em uma nova aba ou nova janela.

_parent:
* Abre o link no frame pai (usado em páginas com frames).

_top:
* Abre o link no frame mais externo (quebra todos os frames e abre na janela completa).

nomeDoFrame:
* Abre o link em um frame específico.

<img src=""> (Image)


Função: usada para incluir uma imagem em uma página HTML.
Uso semântico: Indica que uma imagem deve ser exibida na posição onde a tag é colocada.

Atributo obrigatório: O atributo src especifica o caminho ou URL da imagem.
alt é o texto alternativo, para descrever a imagem.

<del> (Deleted Text)

Função: usada para indicar que um texto foi excluído ou removido.
Uso semântico: Destaca texto que foi removido ou excluído. Frequentemente renderizado com um traço no meio do texto.

<s> (Strikethrough)

Função: usada para aplicar um traço no meio do texto.
Uso semântico: Indica que o texto é incorreto, irrelevante, ou que não é mais válido. Não adiciona significado específico ao conteúdo além de mostrar que está "riscado".

<u> (Underline)

Função: usada para sublinhar o texto.
Uso semântico: No HTML5, indica que o texto é estilisticamente diferente do texto normal, sem necessariamente adicionar ênfase. Por exemplo, pode ser usado para nomes próprios em um texto chinês, ou para marcar erros de ortografia.

<ins> (Inserted Text)

Função: usada para indicar que um texto foi inserido ou adicionado.
Uso semântico: Destaca texto que foi recentemente inserido ou adicionado. Frequentemente renderizado com sublinhado.

<small> (Small)

Função: usada para indicar que o texto deve ser exibido em um tamanho de fonte menor que o texto circundante.
Uso semântico: Indica que o texto é de menor importância, tamanho ou relevância em relação ao texto circundante.

<sup> (Superscript)

Função: usada para colocar o texto em um formato de sobrescrito, posicionando-o acima da linha de base normal.
Uso semântico: Indica que o texto é um expoente, nota de rodapé, ou outra informação relacionada que deve ser exibida acima do texto circundante.

<sub> (Subscript)

Função: é usada para colocar o texto em um formato de subscrito, posicionando-o abaixo da linha de base normal.
Uso semântico: Indica que o texto é um índice, nota de rodapé, ou outra informação relacionada que deve ser exibida abaixo do texto circundante.

<q> (Quote)

Função:Define uma citação curta.
Uso semântico: Envolve texto que representa uma citação curta de outra fonte, frequentemente renderizada entre aspas.

<blockquote> (Block Quote)

Função:
uma citação em bloco.
Uso semântico: Envolve texto que representa uma citação longa de outra fonte, frequentemente renderizada com recuo.
Observação: não é possível inserir dentro da tag <p>, mas o inverso é aceito.

<code> (Code)

Função: Define um trecho de código.
Uso semântico: Envolve texto que representa um fragmento de código de programação, renderizado com uma fonte monoespaçada.

<pre> (Preformatted Text)

Função: 
                Exibe texto
                pré-formatado.
Uso semântico: Preserva o formato do texto, incluindo espaços em branco e quebras de linha, utilizando uma fonte monoespaçada. Observação: não é possível inserir dentro da tag <p>.

<span>

Função: Contêiner genérico em linha para estilização ou manipulação. Para conteúdo in-line
Uso semântico: Não adiciona significado semântico, usado principalmente para aplicar estilos ou scripts a uma parte específica do texto.

<div>

Função:
Contêiner genérico em bloco para estruturação, estilização ou manipulação. Para conteúdo em bloco
Uso semântico: Não adiciona significado semântico, usado principalmente para agrupar conteúdo e aplicar estilos ou scripts.

<nav>

Função:
usada para definir uma seção de navegação em uma página, que contém links para outras partes do site ou da página.
Uso semântico: Indica que o conteúdo dentro da tag é um conjunto de links de navegação, como menus, índices ou barras de navegação.

<ol> (Ordered List)

Função: Cria uma lista ordenada (numerada).
Uso semântico: Itens têm uma ordem específica.

  1. primeiro item
  2. segundo item
  3. terceiro item

<li> (List Item)

Função: Define um item dentro de uma lista.
Uso semântico: Representa um item dentro de uma lista ordenada ou não ordenada.

<ul> (Unordered List)

Função: Cria uma lista não ordenada (com pontos).
Uso semântico: Itens não têm uma ordem específica.

  • primeiro item
  • segundo item
  • terceiro item
